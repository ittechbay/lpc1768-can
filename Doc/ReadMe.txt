测试说明：
    将CAN1与usb转CAN调试器连接连至电脑 CANL-CANL  CANH-CANH，设置CAN波特率为500K，
	插上跳线帽JP15，将开发板串口2（九针串口）与电脑连接，设置波特率为9600。下载程序
	后复位开发板，串口打印信息“*** CAN Test ***”，随后串口输入字符通过CAN输出至上位机
	同时可以通过上位机的USB转CAN调试助手发送字符给开发板通过串口打印出来。
	
CAN滤波设置：
	unsigned int EXP_STD_ID[2]={0x0001,0x0002};			//接收标识符为0x0001和0x0002的标准标识符帧
	unsigned int GRP_STD_ID[2]={0x200,0x210};			//接收标识符在0x200到0x210之间的标准帧标识符
	unsigned int EXP_EXT_ID[2]={0x100000,0x100001};		//接收标识符为0x100000和0x100001的扩展标识符帧
	unsigned int GRP_EXT_ID[2]={0x200000,0x200010};		//接收标识符在0x200000到0x200010之间的扩展帧标识符
	根据程序中的滤波器设置可以看到，当开发板设置滤波后，开发板可以接收到如下几种数据帧：
	1：标准数据帧：ID=0X0001或ID=0X0002
	2：标准数据帧：0X200<=ID<=0X210
	3：扩展数据帧：ID=0X100000或ID=0X100001
	4：扩展数据帧：0X200000<=ID<=0X200010
	通过上位机发送的数据帧需满足如下条件开发板才能接收到，接收到后通过串口打印出来
	用户可以根据自己的需求修改需要滤得的ID
	
usb转CAN调试助手需与usb转CAN调试模块相匹配
